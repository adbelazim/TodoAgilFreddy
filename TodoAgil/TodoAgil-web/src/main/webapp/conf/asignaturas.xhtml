Asignaturas

En esta página puedes adminstar las asignaturas de la aplicación. Se pueden agregar, editar y eliminar asignaturas. También se configuran los sprints y los grupos de sprints para cada una de ellas







Para los grupos de sprint solo se define el nombre, los objetivos y el usuario alumno que hará de Scrum Master. Este último es el encargado de asociar un tablero a esta asignatura en la sección de Tableros.

Importante: Si el grupo es el mismo para todos los sprints, este debe tener el mismo nombre.
